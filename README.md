# Портфолио · Винник Полина

Добро пожаловать в репозиторий моего сайта-портфолио. Это одностраничный сайт, созданный для поступления в Нахимбио лицей БГУ, с целью стать фармацевтом. Сайт представляет мои достижения, увлечения и цели в лаконичном и современном формате.

## 🖥️ Демо

Посмотреть сайт в действии можно по ссылке:  
[https://vpv-ai.github.io/portfolio/](https://vpv-ai.github.io/portfolio/)

## 📋 Содержание портфолио

- **Общая информация**: Имя, средний балл (9.9/10), владение языками.
- **Обо мне**: Краткая автобиография, учебные интересы и профессиональные цели.
- **Достижения**: Список дипломов за 2025-2026 учебный год по олимпиадам (биология, химия, русский язык, физика).
- **Языки и успеваемость**: Детализация уровней английского (B2) и немецкого (A2).
- **Хобби и интересы**: Рисование, чтение, изучение языков.
- **Социальные сети**: Ссылки на Telegram, Instagram и почту.

## ✨ Особенности

- **Две цветовые темы**: Тёмная (по умолчанию) и светлая, переключаемые тумблером.
- **Адаптивный дизайн**: Корректно отображается на всех устройствах.
- **Интерактивность**: Модальные окна для просмотра дипломов и целей.
- **Акценты**: В тёмной теме — голубое свечение, в светлой — красное.

## 🛠️ Технологии

- **HTML5**
- **CSS3** (Flexbox, Grid, CSS-переменные, медиазапросы)
- **JavaScript** (переключение тем, управление модальными окнами)

## 🚀 Локальный запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/vpv-ai/portfolio.git
   ```
2. Откройте файл `index.html` в любом браузере.
